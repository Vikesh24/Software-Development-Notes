## 🎯 Why This Matters  
- Python is a cross platform programming language.
- Python has to be installed to run any python programs.
- There are different methods to set up python on different Operating Systems.
---  

## ⚡ TL;DR (2–3 lines)  

- Windows does not come with Python installed, has to be installed manually.
- Python will be already installed on most of the macOS, but most likely a outdated version.
- Python will be installed in most of the LINUX systems.
  
---  
  
## 🧩 Problem It Solves  
- Make sures python code can be run in the local machine

## 🔬 How It Works (Step-by-Step)  
1.  Check if python is already installed by executing the command python in the terminal or command prompt.
2. If error thrown or if the current version is lower than the latest version, download the latest python installer from https://python.org
